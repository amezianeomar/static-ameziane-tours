# AMEZIANE TOURS 🦅

![AMEZIANE TOURS Banner](images/stork_mosque_portrait.png)

A fully responsive, modern, and immersive multi-page static website built for **AMEZIANE TOURS**, a premium Moroccan tourism agency specializing in authentic cultural encounters, curated guided tours, and expert local insights. 

---

## ✨ Features

- **Modern UI/UX Design**: Striking, premium layout tailored for travel agencies with Glassmorphism elements.
- **Moroccan Aesthetic**: Custom color palette featuring elegant typography and traditional motifs alongside striking imagery.
- **Mobile-First Responsiveness**: Perfect rendering across all devices (Desktop, Tablet, Mobile) using Tailwind CSS Grid and Flexbox.
- **Interactive Elements**: Smooth scrolling, infinite CSS fleet marquee, subtle animations, and refined hover states on buttons and cards.
- **Conversational Booking Wizard**: Highly interactive multi-step booking form evaluating pricing and seamlessly integrating with WhatsApp for direct client inquiries (`order.html`).
- **Production-Ready**: Highly optimized, semantically structured HTML5, utilizing Tailwind CSS via CDN for fast styling.
- **Key Pages & Sections**:
  - **Home (`index.html`)**: Features a sticky navbar, captivating hero section, "Our Story", experiences grid, and an infinite scrolling fleet marquee.
  - **About & Fleet (`about.html`)**: In-depth look at the company narrative with a comprehensive showcase of the vehicle fleet.
  - **Experiences (`experiences.html`)**: Discover curated tourism packages, Sahara camps, Imperial cities, and Atlas Treks.
  - **Booking Wizard (`order.html`)**: Conversational glassmorphism intake form with direct WhatsApp integration.

---

## 🚀 Getting Started

Since this is a static multi-page application built with standard HTML/CSS/JS, no build tools or package managers are required to run the project. 

### Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/amezianeomar/static-ameziane-tours.git
   ```
2. **Navigate into the project directory:**
   ```bash
   cd static-ameziane-tours
   ```
3. **View the website:**
   Simply double-click the `index.html` file to open it in your default web browser.

   *Alternatively, if you prefer using a local server (recommended for testing responsive features and forms):*
   - Using Python: `python3 -m http.server 8000`
   - Using PHP: `php -S localhost:8000`
   - Using Node.js (if `serve` is installed): `npx serve`
